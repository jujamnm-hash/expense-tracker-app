## چۆن بەکاری بهێنم؟

### لە GitHub Pages:
```
https://YOUR_USERNAME.github.io/expense-tracker-app/
```

ناوی `YOUR_USERNAME` بگۆڕە بە یوزەرنەیمەکەی خۆت لە GitHub.

## دیپلۆی کردن

### ١. دروستکردنی ریپۆزیتۆری:
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR_USERNAME/expense-tracker-app.git
git push -u origin main
```

### ٢. چالاککردنی GitHub Pages:
- بڕۆ بۆ Settings → Pages
- لە Source هەڵبژێرە "GitHub Actions"
- چاوەڕێی ٢-٣ خولەک بە

### ٣. بینینی ئەپەکە:
```
https://YOUR_USERNAME.github.io/expense-tracker-app/
```
