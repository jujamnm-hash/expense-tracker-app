# بەدواداچوونی خەرجی 💰

سیستەمێکی پڕۆفیشناڵ و پێشکەوتووی بەدواداچوونی خەرجیە رۆژانەکان و قەرزەکان بە پشتیوانی لە زمانی کوردی.

## ✨ تایبەتمەندیە سەرەکیەکان

### 📊 **داشبۆردی زیرەکانە**
- شیکاری AI و پێشنیارە زیرەکانەکان
- پێشبینی خەرجیەکان بەپێی ناوەندی رۆژانە
- بەراوردی مانگەکان و ترێندەکان
- ئاگادارکردنەوە لە زیادبوونی خەرجی

### 💵 **بەدواداچوونی خەرجیەکان**
- تۆمارکردنی خەرجیە رۆژانەکان بە پۆلێنکردن
- **خەرجیە دووبارەکان** (رۆژانە، هەفتانە، مانگانە)
- **سیستەمی تاگ** بۆ ڕێکخستنی باشتر
- **سێرچ** لە خەرجیەکان
- فلتەر بە جۆر، مانگ و تاگ

### 💸 **سیستەمی قەرز**
- بەدواداچوونی قەرزگرتن و قەرزدان
- زیادکردنی پارەدان و حیسابی ماوە
- پیشاندانی پرۆگرێسی پارەدان
- مێژووی تەواوی پارەدانەکان

### 💰 **بودجەی مانگانە**
- دانانی سنووری بودجە بۆ هەر جۆرێک
- پیشاندانی پرۆگرێس بە ڕەنگ
- ئاگاداری کاتێک لە ٨٠٪ و ١٠٠٪ دەگاتە
- حیسابی ئۆتۆماتیکی خەرجی و ماوە

### 🎯 **ئامانجە داراییەکان**
- دروستکردنی ئامانج بۆ پاشەکەوت
- زیادکردنی پارە بە هێواشی
- پرۆگرێس بار بە ڕەنگە جیاوازەکان
- دیاریکردنی کۆتایی

### 📈 **راپۆرتی پێشکەوتوو**
- چارتی پای بۆ پۆلێنکردن
- چارتی هێڵ بۆ خەرجیە رۆژانەکان
- چارتی بار بۆ بەراوردی جۆرەکان
- ئامارێکی تەواو و وردەکاری
- داونلۆدی راپۆرت بە CSV

### 🎨 **تایبەتمەندیە گشتیەکان**
- **دارک مۆد** 🌙 - پاراستنی چاو لە شەوان
- **٣ زمان**: کوردی، عەرەبی، English
- **٣ جۆری پارە**: دینار، دۆلار، یۆرۆ
- **پاسوۆرد** 🔐 - پاراستنی زانیاریەکان
- **بەکوپ و ڕێستۆر** 💾 - پاشەکەوتی سەلامەت
- **نۆتیفیکەیشنی رۆژانە** 🔔 - یادەوەری زیرەکانە
- **PWA** 📱 - کارکردن وەک ئەپ

## 🚀 دەستپێکردن

### پێداویستیەکان

- Node.js (v18 یان بەرزتر)
- npm یان yarn

### دامەزراندن

```bash
# دامەزراندنی پاکێجەکان
npm install

# کارپێکردنی سێرڤەری گەشەپێدان
npm run dev

# بیلدکردن بۆ بەرهەمهێنان
npm run build

# پیشاندانی بیلدی بەرهەمهێنان
npm run preview
```

سێرڤەری گەشەپێدان لەسەر `http://localhost:3000` دەکرێتەوە.

## 🛠️ تەکنەلۆجیاکان

- **React 18** - کتێبخانەی UI
- **TypeScript** - زمانی پرۆگرامکردن
- **Vite** - بیلد تووڵ
- **Tailwind CSS** - فرەیموۆرکی CSS
- **Zustand** - بەڕێوەبردنی ستەیت
- **Recharts** - کتێبخانەی چارت

- **Lucide React** - ئایکۆنەکان
- **PWA** - پشتیوانی لە Progressive Web App

## 📱 بەکارهێنان

### خەرجیەکان

1. کلیک لەسەر دوگمەی "زیادکردنی خەرجی"
2. بڕ، جۆر، بەروار و وەسف بنووسە
3. خەرجیەکانت بە رۆژ، مانگ و جۆر فلتەر بکە

### قەرزەکان

1. دوگمەی "زیادکردنی قەرز" داگرە
2. جۆری قەرز هەڵبژێرە (قەرزگرتن یان قەرزدان)
3. وردەکارییەکان پڕبکەرەوە
4. پارەدان زیاد بکە و بەدواداچوونی ماوەی قەرز بکە

### راپۆرتەکان

- راپۆرتی مانگانە ببینە بە چارت و ئامارەوە
- شیکاری خەرجیەکان بەپێی جۆر
- چارتی خەرجیە رۆژانەکان
- داونلۆدی راپۆرتەکان بە فۆرماتی CSV

### ڕێکخستنەکان

- چالاک/ناچالاککردنی یادەوەری رۆژانە
- دیاریکردنی کاتی نۆتیفیکەیشن
- بینینی ئاماری بەکارهێنان
- بەڕێوەبردنی زانیارییەکان

## 🌐 دیپلۆی کردن لە GitHub Pages

### هەنگاوەکان:

#### 1️⃣ دروستکردنی ریپۆزیتۆری لە GitHub

```bash
# Initialize git (if not already)
git init

# Add all files
git add .

# Commit
git commit -m "Initial commit - Kurdish Expense Tracker App"

# Add remote (گۆڕینی YOUR_USERNAME بە یوزەرنەیمەکەت)
git remote add origin https://github.com/YOUR_USERNAME/expense-tracker-app.git

# Push to GitHub
git branch -M main
git push -u origin main
```

#### 2️⃣ چالاککردنی GitHub Pages

1. بڕۆ بۆ ریپۆزیتۆریەکەت لە GitHub
2. کلیک لە **Settings** بکە
3. لە لای چەپەوە **Pages** هەڵبژێرە
4. لە ژێر **Build and deployment**:
   - **Source**: هەڵبژێرە "GitHub Actions"
5. پاش چەند چرکەیەک، ئەپەکەت لە لینکێکی وەک ئەمە دەردەکەوێت:
   ```
   https://YOUR_USERNAME.github.io/expense-tracker-app/
   ```

#### 3️⃣ تازەکردنەوەی ئۆتۆماتیک

هەر جارێک push بکەیتە `main` branch، ئەپەکە ئۆتۆماتیک تازە دەبێتەوە!

```bash
# گۆڕانکاری بکە و commit بکە
git add .
git commit -m "گۆڕانکاری زیادکرا"
git push
```

#### 4️⃣ بینینی ئەپەکە

دوای ٢-٣ خولەک، ئەپەکەت ئامادە دەبێت لە:
```
https://YOUR_USERNAME.github.io/expense-tracker-app/
```
